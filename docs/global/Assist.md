# Support

Something broke? Here's how to fix it or who to ping.

## Filing Issues

We track everything on [GitHub Issues](https://github.com/MiamiBeachBots/2026-bot/issues).

### 🐛 Code Bugs
If the robot crashes or doesn't do what you expected:
1. Search existing issues first.
2. Label it `bug` or `urgent`.
3. Include the **Full Error Trace** from the Driver Station console.

### 💬 Requests
Want a new feature or found a typo in the docs?
1. Open an issue.
2. Label it `feature` or `docs`.
